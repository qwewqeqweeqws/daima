# Python_车牌识别系统

## 首先需要知道的
毕业设计基于Opencv的车牌识别系统
- 版本：python3.7.3，opencv4.0.0.21，numpy1.16.2，tkinter和PIL5.4.1.

## 大概演示
由于样本数据来自网络，因此识别率只是看看而已。但清楚的图片还是可以识别出来的。
